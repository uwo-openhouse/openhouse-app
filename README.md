# Description
Mobile app for The University of Westerns Open House Event. 

- yarn (instead of npm)
- EXPO
- React Native
- React Native Navigation
- React Native Elements (for styling)



# Getting Started 

## Getting to know React 101

http://www.reactnativeexpress.com/

## Running the Project

Clone the project.

Install Libs:
`yarn`

Run with:
`expo start`
